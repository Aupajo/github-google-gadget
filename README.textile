h1. GitHub Google Gadget

An iGoogle gadget for displaying GitHub repositories.

"Add to iGoogle":http://www.google.co.nz/ig/adde?hl=en&moduleurl=http://labs.metanation.com/github-google-gadget/legacy.xml&source=imag
